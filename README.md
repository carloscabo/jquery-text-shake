# JQuery textShake v0.1
JQuery plugin that provides some animations to show text

![GitHub Logo](sample.gif)

# Requirements

- JQuery 1.7+

# Usage
```
$(element).textShake(); // Default options
```

Available options.

```
$(element).textShake({
  letterdelay: 60,
  autoplay: true,
  callback: null
});
```
